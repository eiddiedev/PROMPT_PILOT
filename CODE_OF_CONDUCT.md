# Code Of Conduct

## Our Commitment

We want this project to be a welcoming, low-ego space for builders, learners, and contributors of all experience levels. Harassment, exclusion, or dismissive behavior have no place here.

## Expected Behavior

- Be respectful and assume good intent.
- Give feedback on ideas and code, not on people.
- Share context generously, especially when disagreeing.
- Help keep discussions actionable and kind.

## Unacceptable Behavior

- Personal attacks, insults, or discriminatory language
- Harassment, intimidation, or doxxing
- Repeated bad-faith arguing or derailing
- Publishing private information without permission

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please report it privately via the security contact listed in [SECURITY.md](./SECURITY.md).

## Scope

This code of conduct applies within project spaces and in public spaces when someone is representing the project.
